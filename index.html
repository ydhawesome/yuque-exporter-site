Features

导出，但一个细节都不丢

语雀的富文本里藏着图片、公众号卡片、超链接——普通复制粘贴全会丢。这个工具逐一还原。

📝

小记完整导出

分页拉取账号下的全部小记，无数量上限。

  • 正文文字完整保留
  • 图片下载到本地并随文嵌入
  • 书签 / 公众号链接卡片转为可读链接
  • 每条生成独立 Markdown 文件

所有小记汇总成一张表，图片直接嵌进单元格。

  • 纯 JS 生成，无需 Python/Office

How it works

三步，几分钟搞定

不需要任何技术背景，下载即用。

下载并打开

从 GitHub Releases 下载便携版 exe，双击运行，输入语雀账号密码登录。

选择导出目录

指定一个本地文件夹，用来存放导出的 Markdown、图片和 Excel。

一键导出

在「小记」或「知识库」标签页点击导出，进度实时显示，完成后一键打开文件夹。

🛡️

你的数据，只属于你

这是一个桌面应用，不是网站。它在你自己的电脑上直接向语雀官方接口请求数据，账号密码不会上传到任何第三方服务器，也不写入代码或日志。代码完全开源，每一行都可审计。

免费、开源、本地运行。你的笔记值得一份可靠的备份。
